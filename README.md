## 🚀 Installation  
1️⃣ **Clone the repository:**  
   ```sh
https://github.com/thasneem9/evolon.git
```
2️⃣ **Navigate to the project folder:**  
  ```sh
cd evolon
```
3️⃣  **Install dependencies:**
```sh
npm install
```
 4️⃣ **Run Project:**
```sh
npm run dev
```
